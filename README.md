
# Delivery Record Management System

A normalized DBMS project for managing delivery operations using Oracle SQL and PL/SQL.

## Features
- Customer, order, payment, assignment, and complaint management
- ER modeling and 3NF normalization
- PL/SQL procedures, triggers, and cursors
- Transaction management with COMMIT/ROLLBACK
- Reporting queries using joins, aggregates, and views

## Technologies Used
- Oracle DBMS
- SQL
- PL/SQL

## Database Modules
- CUSTOMER
- ORDER_DETAILS
- DELIVERY_AGENT
- ASSIGNMENT
- PAYMENT
- COMPLAINT

## Concepts Implemented
- ER Modeling
- Relational Schema Design
- 3NF Normalization
- Stored Procedures
- Triggers
- Cursors
- ACID Transactions
